测序Reads比对

  • 测序Reads原始数据下载, 查阅关联群体甲基化的文章:

    Xu J, Chen G, Hermanson PJ, Xu Q, Sun C, Chen W, et al. Population-level analysis reveals the widespread occurrence and phenotypic consequence of DNA methylation variation not tagged by genetic variation in maize. Genome Biol. 2019;20:243.

  • 每个材料的甲基化测序reads要比对到各自参考基因组, HZS的参考基因组在: /jbyan/stgui/work/09_ZEAMAP_database/00_CNGBDB_V1.0/01_Offical_data/01_genomics/01_genomes/08_Zea_mays_ssp_mays/HZS/Zea_mays_ssp_mays.HZS.v1.0.genome.fa.gz

  • 比对工具我用的是bsmap(集群后来数据迁移, 原始环境中的软件删掉了, 请自行安装), 比对完后的bam需要sort, 具体命令如下:

bash

ref=/path/to/your/ref.fa
r1=/path/to/read1.fq.gz
r2=/path/to/read2.fq.gz
pre="output-prefix"
cpu=4
if [ ! -s "${pre}.bsmap.mkdup.bam" ];then
    if [ ! -s "${pre}.bsmap.sort.bam" ];then
        if [ ! -s "${pre}.bsmap.bam" ];then
            bsmap -a $r1 -b $r2 -d $ref -q 20 -f 5 -p $cpu -v 0.05 -s 16 -S 1 -n 0 |\
            samtools view -@ $cpu -Sb -o ${pre}.bsmap.bam
            if [ $? -ne 0 ];then echo "bsmap failed: Non-zero exit" >&2; rm -f ${pre}.bsmap.bam; exit 1;fi
        fi
        sambamba sort -t $cpu -o ${pre}.bsmap.sort.bam ${pre}.bsmap.bam
        if [ $? -ne 0 ];then echo "sort bam failed: Non-zero exit" >&2; rm -f ${pre}.bsmap.sort.bam*; exit 1;fi
    fi
    sambamba markdup -t $cpu ${pre}.bsmap.sort.bam ${pre}.bsmap.mkdup.bam &&\
    rm -f ${pre}.bsmap.bam ${pre}.bsmap.sort.bam*
fi

bash

  • 如果有多个重复的reads, 每个重复的bam还要合并, 用samtools merge即可

鉴定位点的甲基化水平

用MethylDackel, 具体命令/参数如下:

bash

ref=/path/to/ref.fa
bam=/path/to/bsmap.sort.bam
cpu=4
MethylDackel extract -q 20 -p 5 -d 3 -@ $cpu --CHG --CHH $ref $bam

鉴定高和低甲基化区间

鉴定高低甲基化区间的标准与2020年NAMScience文章中的标准保持一致:

To identify UMRs, reference genomes were segmented into 100-bp intervals. Intervals lacking at least four covered CHG-context cytosines (CHGs) were discarded. Coverage was calculated on a per-cytosine basis and summed over each interval, and any interval with less than 20 reads covering CHGs was discarded. Intervals with methylation of greater than 20%, calculated using the weighted methylation formula (Schultz et al. 2012), were also discarded. This was repeated on 20bp sliding increments, and all overlapping intervals or intervals separated by only 20 bp were merged to define larger UMRs. UMR edges were then trimmed such that their boundaries were defined by CHGs with less than or equal to 20% methylation. At this stage UMRs that overlapped with blacklisted regions (identified based on abnormally high coverage of the 150nt paired end Illumina reads that were used in each genome’s assembly) were discarded. This process was repeated using CG/CHG methylation combined rather than CHG methylation alone and both sets of UMRs were merged. Finally, UMRs that were shorter than 150 bp in length were discarded.

我写了一个脚本用来流程化以上操作:

源码: /jbyan/stgui/sftw/gst_scripts/gst_hlmr.sh

需要预先安装以下软件才可以使用: bedtools, csvtk, bgzip

用法如下(默认参数就是我们甲基化数据的参数):

bash

------------------------------------------------------------
Get LMR and HMR from input methylation bedgraph
------------------------------------------------------------
Dependence: bedtools, csvtk, bgzip
------------------------------------------------------------
USAGE:
    bash gst_hlmr.sh [OPTIONS]

OPTIONS: ([R]:required  [O]:optional)
# common; I/O
    -h, --help                          show help and exit.
    -i, --in            <str>   [R]     input file in bedgraph format
    -o, --out           <str>   [O]     output prefix name (default: prefix of --in file)
# sliding window
    -w, --window        <int>   [O]     set sliding window size in bp (default: 100)
    -s, --step          <int>   [O]     set step size in bp (default: 20)
# site filtering
    --min_cd            <int>   [O]     the minimum cytosine supported read number to support methylation/un-methylation (default: 3)
    --min_td            <int>   [O]     the minimum cytosine supported read number to support methylation/un-methylation (default: 3)
# window filtering
    --min_rd            <int>   [O]     the minimum number of supported reads in the region (default: 20)
    --min_nc            <int>   [O]     the minimum number of cytosine sites in the region (default: 4)
    --min_mrl           <int>   [O]     the minimum length of methylation region in bp (default: 150)
    --lmr             <1-100>   [O]     set LMR methylation level cut-off from 0 to 100 (%) (default: 20)
    --hmr             <1-100>   [O]     set HMR methylation level cut-off from 0 to 100 (%) (default: 80)
# misc
    --keep                              Kept tmp files
------------------------------------------------------------

提取序列

我也写了一个脚本批量根据甲基化区间提取序列, 我们提取序列的标准是:

  1. 按照固定长度滑窗

  2. 如果区间小于滑窗长度, 双端补齐N

提序列脚本: /jbyan/stgui/work/05_MethylML/01_Models/00_training_data/00_seqs_and_labels/get_win_seq.sh

用法:

bash

NOTE:
    prefix should be [REF]__[Tissue]__[MethyType], eg:
    "B73__root__CHG"
